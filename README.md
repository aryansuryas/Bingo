## вңЁ What Is This?

> **BINGO** is a premium single-page browser game вҖ” no installs, no frameworks, no dependencies. Just open `index.html` and play.

You and an AI bot each receive a **randomized 5Г—5 board** filled with numbers 1вҖ“25. Numbers are called by turns вҖ” yours when you click, the bot's when it "thinks." Any number called gets **marked on both boards simultaneously**. First to complete **5 full lines** (rows, columns, or diagonals) wins.

---

## рҹҺ® How to Play

```
1. Open index.html in any modern browser
2. Your board is on the LEFT вҖ” click any number to call it
3. That number is marked on BOTH boards instantly
4. Wait 1вҖ“2 seconds вҶ’ the Bot makes its move
5. First to complete 5 lines вҶ’ BINGO! рҹҺү
```

## рҹ§  Bot AI вҖ” How It Thinks

The bot isn't random. It uses a **weighted scoring strategy**:

```javascript
// For each available number, score it by how many
// near-complete lines it belongs to on the bot's board:

score += alreadyMarkedВІ // quadratic reward for nearly-done lines

// Plus a touch of randomness to keep it beatable:
score += Math.random() * 1.5
```

This means the bot:
- **Aggressively closes** lines that are 3вҖ“4 deep
- **Occasionally surprises** you with unexpected picks
- **Is beatable** вҖ” strategy and luck both matter
